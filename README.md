# 🧠 Insurance Appetite Matching Platform

> An intelligent platform to match insurance appetite based on industry and coverage requirements.

---

## 📌 Overview

This project is an intelligent insurance appetite matching platform designed to help users quickly identify suitable insurance options based on specific industries and insurance types.

It improves decision-making efficiency by providing structured matching results and clear appetite classification.

---

## ✨ Features

- 🔎 **Smart Search Functionality**  
  Search by industry and insurance type to get accurate matching results

- 📊 **Structured Matching Results**  
  Clearly organized results for easy comparison and understanding

- 🧠 **Appetite Classification**  
  Categorizes results into:
  - High Appetite  
  - Medium Appetite  
  - Low Appetite  

- 📈 **Search Summary Insights**  
  Displays matching statistics and overview

- ⚡ **Fast & Responsive UI**  
  Smooth user experience with clean layout
